# NASA Exoplanet Research

This repository contains a reproducible research workflow built on the NASA Exoplanet Archive. It re-estimates the exoplanet radius valley for close-in small transiting planets and ranks current TESS Objects of Interest (TOIs) by follow-up value.

## Overview

The project answers three practical research questions:

1. How does the radius valley shift with stellar irradiation in the latest confirmed transiting-planet sample?
2. Is there a detectable metallicity dependence in the radius valley?
3. Which unconfirmed TOIs would provide the highest demographic value if they are validated and characterized?

The workflow uses the NASA Exoplanet Archive `PSCompPars` table for confirmed planets and the `TOI` table for TESS planet candidates.

## Current Results

Using the snapshot analyzed in this repository:

- Confirmed sample: 2,896 planets
- TOI candidate sample: 854 candidates
- Fitted relation:
  `log10(R_valley / R_earth) = 0.030 + 0.108 * log10(S / S_earth)`
- Valley radius rises from about `1.37 R_earth` in the lowest irradiation bin to `2.16 R_earth` in the highest bin
- A metallicity trend is visible, with the valley shifting upward from `1.71 R_earth` to `2.01 R_earth` across metallicity tertiles

These results should be treated as a reproducible empirical analysis, not a fully selection-corrected population model.

## Repository Structure

- `scripts/fetch_nasa_exoplanet_data.sh`
  Downloads fresh CSV snapshots from the NASA Exoplanet Archive using TAP queries.
- `src/nasa_exoplanet_research.py`
  Cleans the data, estimates the radius valley, ranks TOI candidates, and generates plots and summary files.
- `data/raw/`
  Raw CSV snapshots downloaded from NASA.
- `data/processed/`
  Cleaned analysis tables generated by the pipeline.
- `results/`
  Figures, candidate rankings, and a machine-readable research summary.
- `REPORT.md`
  Human-readable summary of the current run and its main findings.

## Quick Start

### 1. Fetch the latest NASA data

```bash
bash scripts/fetch_nasa_exoplanet_data.sh
```

### 2. Run the analysis

```bash
python3 src/nasa_exoplanet_research.py
```

If you are running in a restricted or headless environment, this is the safest invocation:

```bash
MPLCONFIGDIR=/tmp/mplconfig python3 src/nasa_exoplanet_research.py
```

## Method Summary

- Filter confirmed transiting planets to a compact small-planet regime
- Estimate the radius valley with kernel density minima between the super-Earth and sub-Neptune peaks
- Fit a flux-dependent valley trend
- Measure a metallicity-split valley shift
- Rank TOIs with a composite score based on novelty, brightness, transit depth, and proximity to the predicted radius valley

## Notes

- Data access is intentionally handled through `curl`, and the Python analysis reads local CSV snapshots only.
- The current scoring system is designed for follow-up triage, not for formal validation probability estimates.
- This repository is suitable for a public GitHub release as a compact, reproducible astronomy research project.
